type: feedback
---

Всегда разбивать задачи на этапы: анализ → подробный план → поэтапное выполнение мелкими итерациями.

**Why:** Пользователь хочет контролировать процесс, видеть план до начала работы, получать стабильный код после каждого шага.

**How to apply:**
1. Прочитать `.claude/memory/` файлы, релевантные задаче
2. Прочитать конкретные файлы кода
3. Войти в plan mode, составить план
4. Показать план, дождаться подтверждения
5. Выполнять по одной итерации, lint после каждой
6. Сообщать прогресс
