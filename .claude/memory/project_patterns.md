## Создание новой страницы

1. Route: создать файл в `src/routes/_private/feature.tsx` (или `feature.lazy.tsx` для lazy loading)
2. Page: создать `src/pages/feature/Feature.tsx`
3. VM hook: `src/pages/feature/hooks/useFeatureVM.tsx` — связывает store + API
4. Store (если нужен): `src/store/feature/FeatureStore.ts`
5. TanStack Router автоматически обновит `routeTree.gen.ts`

## Создание нового Store

```typescript
import { makeAutoObservable } from "mobx";
import { IStoreService } from "@force-dev/react";

@IStoreService({ inSingleton: true })
export class FeatureStore {
  private _holder = new EntityHolder<FeatureDto>();

  constructor() {
    makeAutoObservable(this);
  }

  get data() { return this._holder.d; }
  get isLoading() { return this._holder.isLoading; }

  async load(id: string) {
    await this._holder.loadData(() => apiService.getFeature(id));
  }

  dispose() {
    this._holder.clear();
  }
}
```

## Socket Subscription Pattern

```typescript
// В VM hook или store
useEffect(() => {
  const unsub = wgSocketService.subscribeServer(serverId, {
    onStats: (payload) => statsStore.addDataPoint(payload),
    onStatus: (payload) => serverStore.updateStatus(payload),
  });
  return unsub;
}, [serverId]);
```

## Form Pattern (React Hook Form + Zod)

```typescript
const schema = z.object({
  name: z.string().min(1, "Required"),
  endpoint: z.string().optional(),
});

type FormValues = z.infer<typeof schema>;

const form = useForm<FormValues>({
  resolver: zodResolver(schema),
  defaultValues: { name: "", endpoint: "" },
});

return (
  <Form {...form}>
    <InputFormField name="name" label="Name" />
    <InputFormField name="endpoint" label="Endpoint" />
    <Button type="submit">Save</Button>
  </Form>
);
```

## View Model Pattern

```typescript
// src/models/server/Server.model.ts
export class ServerModel extends DataModelBase<WgServerDto> {
  get statusLabel() { return this.data.status; }
  get shortPublicKey() { return this.data.publicKey?.slice(0, 8) + "..."; }
  get createdAtDate() { return new Date(this.data.createdAt); }
}
```

## Component Pattern (Radix + Tailwind)

Компоненты в `src/components/ui/` используют Radix primitives + Tailwind:
```typescript
import * as DialogPrimitive from "@radix-ui/react-dialog";

export const Modal = ({ children, ...props }) => (
  <DialogPrimitive.Root {...props}>
    <DialogPrimitive.Portal>
      <DialogPrimitive.Overlay className="fixed inset-0 bg-black/50" />
      <DialogPrimitive.Content className="fixed left-1/2 top-1/2 ...">
        {children}
      </DialogPrimitive.Content>
    </DialogPrimitive.Portal>
  </DialogPrimitive.Root>
);
```

## Правила

- **`src/api/api-gen/`** — auto-generated, NEVER edit. `yarn generate:api`
- **`routeTree.gen.ts`** — auto-generated by TanStack Router, NEVER edit
- **Path aliases** обязательны: `~@api`, `~@store`, `~@components`, `~@models`, `~@core`
- **Stores** — singletons через IoC, `@IStoreService({ inSingleton: true })`
- **View models** — обёртки над DTO, computed properties, НЕ бизнес-логика
- **Forms** — React Hook Form + Zod, компоненты из `src/components/ui/form/`
- **Socket** — подписки через `wgSocketService`, НЕ напрямую через transport
- **Стили** — только Tailwind классы + CSS variables, НЕ inline styles, НЕ CSS-in-JS
- **Permissions** — проверять через `usePermissions().hasPermission()`, не хардкодить роли
