## Architecture

### Stack
React 19 + TypeScript + Vite 6 + TanStack Router + MobX + Tailwind CSS 4 + Radix UI + Socket.IO client + Axios

### Key Rules
- `src/api/api-gen/` — **auto-generated from Swagger, NEVER edit manually** (`yarn generate:api`)
- `src/routeTree.gen.ts` — **auto-generated by TanStack Router, NEVER edit**
- **Never import from @force-dev** — all infrastructure is in `@common/`
- Path aliases: `@api`, `@store`, `@components`, `@models`, `@core`, `@common`, `@socket` (без `~` префикса)

### IoC (Dependency Injection)
Internal DI in `src/common/ioc/` (inversify-based). Pattern:
```ts
export const IMyService = createServiceDecorator<IMyService>();
@IMyService({ inSingleton: true })
class MyService { constructor(@IOther() private other: IOther) {} }
const useMyService = iocHook(IMyService);
```

### Routing (TanStack React Router)
File-based routing from `src/routes/`. Protected routes in `_private/` check `auth.isAuthenticated`.

### State Management (MobX)
Singleton stores via IoC. Data holders from `src/common/store/holders/`:
`EntityHolder`, `PagedHolder`, `InfiniteHolder`, `CollectionHolder`, `MutationHolder`, `PollingHolder`, `CombinedHolder`.

### API Layer
Abstract `HttpClient` (generated) → `ApiService` implements `instancePromise()`.
`ApiResponse<R, E>` = `{ data?, error? }`. 401 → refresh + retry. `QueryRace` for dedup.

### WebSocket (Socket.IO)
Auto-reconnect, emit queue, persistent listeners. JWT auth in handshake.
Events: `wg:server:stats`, `wg:peer:stats`, `wg:stats:overview`, `wg:peer:active`, `wg:server:status`.

### Permissions
`hasPermission()` with wildcard matching (`chat:view` → `chat:*` → `*`).
`canAccess()` = admin bypass OR permission check.

### Theming
CSS variables in `src/styles/{light,dark}-theme.css`. ThemeProvider in `src/core/theme/` with localStorage.
