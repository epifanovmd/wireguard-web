## Workflow

При получении любой задачи — следуй этому процессу:

### 1. Анализ
- Прочитай `.claude/memory/` файлы, релевантные задаче
- Прочитай конкретные файлы кода, которые будут затронуты

### 2. План
- Войди в plan mode, составь пошаговый план
- Разбей на мелкие итерации
- Покажи план, дождись подтверждения

### 3. Выполнение
- По одной итерации за раз

- Сообщай прогресс

### 4. Проверка
- `yarn build`
- Кратко резюмируй

## Architecture

### Stack
React 19 + TypeScript + Vite 6 + TanStack Router + MobX + Tailwind CSS 4 + Radix UI + Socket.IO client + Axios

### Key Rules
- `src/api/api-gen/` — **auto-generated from Swagger, NEVER edit manually** (`yarn generate:api`)
- `src/routeTree.gen.ts` — **auto-generated by TanStack Router, NEVER edit**
- Path aliases: `~@api`, `~@store`, `~@components`, `~@models`, `~@core`, `~@service`, `~@theme`

### Routing (TanStack React Router)
File-based routing from `src/routes/`. Protected routes in `_private/` check `auth.isAuthenticated`.
Route guards via `beforeLoad()`. Lazy loading via `*.lazy.tsx`.

### State Management (MobX)
Singleton stores bound via IoC (inversify): `@IStoreService({ inSingleton: true })`.
Data holders from `@force-dev/utils`: `EntityHolder`, `PagedHolder`, `MutationHolder`.

### API Layer
Axios with JWT interceptors. Auto-generated from Swagger (`src/api/api-gen/`).
401 → refresh token + retry. Base URL from `VITE_BASE_URL`.

### WebSocket (Socket.IO)
Auto-reconnect, emit queue, persistent listeners. JWT auth in handshake.
Events: `wg:server:stats`, `wg:peer:stats`, `wg:stats:overview`, `wg:peer:active`, `wg:server:status`.

### Component Library
97+ Radix-based, Tailwind-styled components in `src/components/ui/`.
Forms: React Hook Form + Zod validation.

### Theming
CSS variables in `src/styles/{light,dark}-theme.css`. ThemeProvider with localStorage.
`.dark` class on `<html>` for dark mode.

### Permissions
`EPermissions` enum from API. Frontend `hasPermission()` with wildcard matching.
Route-level guards + component-level conditional rendering.
